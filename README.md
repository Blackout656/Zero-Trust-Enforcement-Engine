#  Continuous Zero-Trust Authentication & Perimeter Enforcement Engine

## 🎯Architectural Overview
A responsive full-stack security orchestration engine that implements true **Continuous Authentication** at the perimeter. Instead of relying on static one-time logins, this system re-evaluates risk parameters on *every single transaction* using multi-factor contextual telemetry, executing automated micro-segmentation and edge firewall isolation drops when anomalies cross critical thresholds.



## Core Features
* **Impossible Travel Velocity Detection:** Calculates mathematical timestamp deltas between consecutive requests to identify credential theft.
* **Automated Incident Isolation (SOAR):** Instantly generates edge network firewall rules (`iptables`) to drop compromised hardware signatures.
* **Dynamic Access Segmentation Matrix:** Routes requests seamlessly between `FULL_ACCESS`, `RESTRICTED_SANDBOX`, and `ACCESS_DENIED` profiles.

## Setup & Interactive Testing
Because this application is built as a self-contained web platform, it requires zero intricate server dependencies.
1. Download or clone this repository.
2. Open `index.html` directly in any modern enterprise web browser.
3. Use the injection panel to simulate malicious DB queries or travel velocity violations to watch the perimeter containment tables update in real time.
